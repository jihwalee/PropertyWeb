# nai_propertree
NAI Propertree에서 사용하는 신사업 웹서비스.

- 로그인 및 회원가입
- GoogleMap 연동을 통한 부동산 정보 출력


## Prerequisites

1. Install [node.js](http://nodejs.org), [MongoDB](http://mongodb.com) properly.
2. Install npm packages.
``` bash
$ npm install -g bower
```

## Start the server
``` bash
$ node server.js
```
